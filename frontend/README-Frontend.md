# EggPrices NZ - Frontend Only Demo

A pure frontend egg price comparison platform for New Zealand, designed to work with static hosting like Netlify.

## Features

✅ **Complete Price Database**: 86 egg products from 3 major NZ supermarkets  
✅ **Real-time Comparison**: PAK'nSAVE, Countdown, and New World prices  
✅ **Interactive Map**: Store locations with distance calculations  
✅ **User Authentication**: Demo login/signup system  
✅ **Premium Features**: Free trial and premium subscription simulation  
✅ **Mobile Responsive**: Works on all devices  
✅ **No Backend Required**: Pure frontend, perfect for Netlify  

## Quick Start

1. **Clone or download** this repository
2. **Open any HTML file** in a web browser or serve with a local server
3. **No installation required** - it's pure HTML, CSS, and JavaScript

## File Structure

```
├── index.html          # Main dashboard

```
